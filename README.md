## 新增页面：沥青吨数计算器（asphalt_calculator）

路径：`/asphalt_calculator`

该页面用于计算沥青铺面所需的总吨数。用户可输入长、宽、厚度和压实沥青单位重量，支持常用单位（米、英寸、吨、千克），自动换算并输出总吨数。页面专为"沥青计算器"等 SEO 关键词优化。
